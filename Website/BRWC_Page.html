Battle Royale with Cheese is a hectic food-based shooter

Key features

Dynamic shop system

Run between 3 different restaurants that spawn weapons for taking down foes

Multiple weapons

Use soda can grenades, pizza shurikens, and a Sriracha sauce flame thrower to keep yourself alive

Relentless wave-based enemies

Try your best to survive a gauntlet of angry customers who want to take you down

Responsive movement

Run, jump, and dodge foes as you try to get to the next weapon pickup

The Game

Twin Pines Mall is having their annual Black Friday blowout event and you want to be first to all the hottest deals.
Unfortunately, a crowd of angry customers had the same idea as you! In order to get the first spot in all the lines,
you must defeat waves of customers while collecting food from nearby restaurants to use as ammunition.

Controls

Movement - WASD
Jump - Space
Run - Shift
Shoot - Left mouse button
Pick up weapons by walking up to open restaurant counters

The Tech

BRWC runs on The Cheese Engine, an entity-component based engine built using OpenGL. The game uses the following technologies:

Visuals

  • Shadow Maps
  • Particle effects
  • Toon shading
  • Bloom
  • HUD

    • Gameplay

  • Axis-alligned bounding volume collisions
  • Enemy pathfinding
  • Hierarchical animations

    • Other

  • Octree spatial optimization
  • JSON file level loading
  • View frustum culling

    • Dependencies

  • GLFW
  • FMOD
  • GLM
  • Assimp

    • Source code is avaliable online at https://github.com/jaafersheriff/BattleRoyale
